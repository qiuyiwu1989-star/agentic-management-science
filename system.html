01_ // blueprint · 6×5 矩阵

智能体时代企业重构总图

AMS 不是一组零散的 AI 管理工具，而是一张企业在智能体时代的系统重构地图。 六大重写回答"企业的哪些基本命题被重写"； 五阶段跃迁标定"企业从工具使用到自进化组织的成熟度位置"； 三治理神经贯穿全程，确保在意图对齐、边界可控、主权清晰的前提下持续进化。

AMS = 六大重写 × 五阶段跃迁 × 三治理神经

02_ // knowledge map · 六个重写 + 治理神经

六个重写模块，一条共同问题链。

每一个模块回答一个独立的根本问题，它们组合在一起，重写了智能体时代的企业管理学。 顺序读一遍 = 每个模块的完整逻辑链。

模块一 竞争重写 · 看世界 → 看优势 → 做选择 → 定路径 → 重估值 → 变收入
模块二 组织重写 · 形态 → 架构 → 权责 → 岗位 → 安全 → 文化
模块三 产品重写 · 定位 → 需求 → 设计 → 验证 → 评估 → 信任 → 架构
模块四 系统重写 · 评测 → Skill → 调试 → 架构 → 生命周期 → 评测层 → 回归
模块五 价值重写 · 运营成熟度 → 指标 → 绩效 → 成本 → ROI → 风险
模块六 人类重写 · 能力等级 → 角色迁移 → 能力结构 → 核心能力 AC6
