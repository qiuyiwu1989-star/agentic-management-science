# 智能体管理学 · Agentic Management Science

> 当执行可以被智能体承担，企业管理学的所有基本假设——竞争结构、组织形态、产品逻辑、系统方法、价值评估、人类定位——都需要被重写一遍。

**造物云思想底座 · v1.0**

## 📚 项目结构

```
.
├── index.html              # 主页：体系全景 + 40 框架索引
├── knowledge-map.html      # 知识图谱
├── patterns.html           # 模式库
├── resources.html          # 资源库
├── framework/              # 40 个框架 + 治理神经
│   ├── F00.html ~ F36.html # 36 个核心框架
│   └── G01.html ~ G03.html # 3 个治理神经
└── assets/
    ├── ams.css             # 站点样式
    └── ams.js              # 站点交互
```

## 🚀 本地预览

直接用浏览器打开 `index.html` 即可，无需构建步骤。

或启动一个本地静态服务器：

```bash
# Python 3
python3 -m http.server 4321

# Node (需安装 npx)
npx serve .
```

然后访问 http://localhost:4321
